# サンプルシナリオ：緑色の手と密室の罪

## 1. イントロダクション

- **場所:** 夜中の「緑の間」
- **状況:** 毒の矢が持ち込まれた伯爵の部屋に散らばっている
- **プレイヤーの役割:** 評議会から派遣された「審判者」。30分後までに犯人を特定しなければならない

---

## 2. 登場人物（NPC）

### ① エレナ（Elena）
- **音声特性:** 落ち着いた、低めの女性の声
- **目的:** 犯行を隠し、妹と共に逃げ出すこと
- **キーワードへの反応:**

| キーワード | 反応 |
|-----------|------|
| 「毒」 | 「それは私...いや、何でもありませんわ！」（激しい動揺） |
| 「妹」 | 「妹は関係ありません、彼女はただの人間です」 |

### ② ヴィクター（Victor）
- **音声特性:** 若くて少し荒れた男性の声
- **目的:** 友人を返すための遺産を確保すること、殺人の罪を洗い流すこと
- **キーワードへの反応:**

| キーワード | 反応 |
|-----------|------|
| 「友人」 | 「あの男のために今こうなのだから、殺しの動機になると？」 |
| 「毒の手袋」 | 「...っ！あれはただの護身用だ！」 |

---

## 3. ゲームの進め方（音声トリガー）

```
1. [音声入力]   「ヴィクターが手の手袋を見ている」
        ↓
2. [システム判定] search_action(target="Victor", item="gloves")
        ↓
3. [GM応答]    「ヴィクターの手袋には、微かに毒の粉が付着しています」（証拠A解禁）
        ↓
4. [NPC変化]   ヴィクターのプロンプトが「言い訳モード」に移行
```

---

## 4. 解決フェーズ（判定ロジック）

プレイヤーが以下の3要素を音声で宣言したら正解と判定する：

| 要素 | 正解 |
|------|------|
| 犯人 | エレナ |
| 制約（凶器） | ヴィクターが忌み捨てた毒の矢 |
| 動機 | 妹を市街鬼化から守る |
